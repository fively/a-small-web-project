# a-small-web-project
一个小的web项目

##### Description
打包构建：webpack
技术栈：React


##### Install

```
pnpm install -w
```

##### Start

启动方式：pnpm dev <app_name>:[<config>]

```
pnpm dev core-web
或 
pnpm dev core-web:dashboard
```

##### Build

打包方式：pnpm build <app_name>:[<config>]

```
pnpm build core-web
```

注：pnpm启动单个项目时，语句有点长，所以才有shell脚本方式做了简化（自看shell指令写的sh文件，有点low。至于为啥这么干，就觉得用的顺手）
